# Futuristic Disk Scheduling Simulator

A high-performance, interactive, sci-fi themed disk scheduling algorithm simulator built with Vanilla JavaScript and Vite.

- **6 Algorithms**: FCFS, SSTF, SCAN, LOOK, C-SCAN, C-LOOK.

## üß† algorithms Implemented

- **FCFS**: First-Come, First-Served.
- **SSTF**: Shortest Seek Time First.
- **SCAN**: Elevator Algorithm (sweeps end-to-end).
- **LOOK**: Adaptive SCAN (sweeps only as far as needed).
- **C-SCAN**: Circular SCAN (one-way sweep).
- **C-LOOK**: Adaptive Circular SCAN.
